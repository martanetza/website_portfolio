Exmouth View Hotel

Together with my group we created a new website of Exmouth View Hotel. The old website of the hotel needed to be redesigned, because it was old fashion, messy, overloaded with information and hard to navigate. Following the principals of design, we created a new website that is better organized, more user friendly and looks more modern. As working on the assignment I was mostly focused on coding. However, I also took part in conceptual work, where we were planning, what the website should look like, so I have learned a lot about communication and design as well.

Interaction Design

I coded the website using HTML5, CSS and Grid Layout Module. The biggest challenge for me in that case was to react quickly to groups decisions on website layout. In fact we changed a lot in the concept of layout short before the deadline, so I had little time to recode the website again. It was a little bit stressful but I am not complaining. I am really happy about that experience. I have learned that it is better to have a code well and logically structured, so that all the changes can be easily and quickly applied. I guess everyone expect coders to be fast, so I hope I can be better organized and more efficient in future.
