Bauhaus Style

I created a simple responsive website about Bauhaus Style. The website consist of four pages. Main page contains a short introduction. Style page is dedicated to the story of Bauhaus style. In design page I describe principals of composition and layout of the website. Finally, profile page presets few information about me.

Interaction

I collected the information using secondary (desk) research methods. Based on that I prepared a mood board, where I included keywords and pictures that inspired me to create the website. After creating mobile version of the website I conducted trunk test. The test showed that there was a bunch of problems that needed to be solved. First of all students had problems with answering the question, what the website is about. Moreover, it was hard for them to navigate within each website. The website seemed to bit overloaded with information. I tried to solve some of the problems while working on the desktop and tablet version.

My aim was to create a website that is clean, aesthetic and easy to use. I wated to create a consistent layout with content of information that is logically organized. In order to do that, I started with creating sketches, digital frameworks and prototypes. Next step was to prepare color scheme, typography and stile tile.

Logo design

The website is created by using HTML5 and CSS. It is responsive, which in this case means that it is compatible with large, medium and small devices. While designing the website I was using the CSS Grid Layout Module, that turned out to be a very practical and useful tool. The Grid Layout allowed me to change positions of the content easily, so that it could be quickly adjust to different devices. Although Grid Layout worked fine for me in that case, I still had some issues with sizes of items displayed in the website. I would like to improve that if future.
